ส่วนที่ 1 — แนวคิดโมล (Mole)
★สูตรโมลทั้งหมด (ต้องท่องจำ!)
$$n = \frac{g}{MW} = \frac{N}{6.022 \times 10^{23}} = \frac{V \text{(STP)}}{22.4}$$

• n = จำนวนโมล (mol)

• g = มวล (กรัม)

• MW = มวลโมลาร์ (g/mol)

• N = จำนวนอนุภาค (อะตอม/โมเลกุล)

• $N_A$ = $6.022 \times 10^{23}$ (เลขอาโวกาโดร)

• V(STP) = ปริมาตร (ลิตร) ที่ STP (0°C, 1 atm) → แก๊ส 1 mol = 22.4 L

🥑 เลขอาโวกาโดร vs อาโวคาโด! $6.022 \times 10^{23}$ ตั้งชื่อตามนักวิทยาศาสตร์ Amedeo Avogadro — ไม่ใช่ผลไม้ 😄 ตัวเลขนี้ใหญ่แค่ไหน? ถ้ามีโมลของเม็ดทรายจะปิดพื้นผิวโลกได้หนา 9 กิโลเมตร!
★สรุปปริมาณสาร 1 โมล NH₃
NH₃ จำนวน 1 โมล (n = 1):

• จำนวนโมเลกุล = $6.022 \times 10^{23}$ โมเลกุล

• จำนวนอะตอม N = $1 \times 6.022 \times 10^{23}$ อะตอม

• จำนวนอะตอม H = $3 \times 6.022 \times 10^{23}$ อะตอม

• จำนวนอะตอมรวม = $4 \times 6.022 \times 10^{23}$ อะตอม

• มวล N = 14 กรัม, มวล H = 3 กรัม, มวลรวม = 17 กรัม

• ปริมาตรที่ STP = 22.4 ลิตร

22คาเฟอีน C₈H₁₀N₄O₂ จำนวน 0.5 โมล
✦ เฉลย
MW คาเฟอีน = (8×12) + (10×1) + (4×14) + (2×16) = 96 + 10 + 56 + 32 = 194 g/mol
1) จำนวนโมเลกุล = $0.5 \times 6.022 \times 10^{23}$ = $3.011 \times 10^{23}$ โมเลกุล
2) โมลอะตอม C = $0.5 \times 8$ = 4 mol, H = $0.5 \times 10$ = 5 mol, N = $0.5 \times 4$ = 2 mol, O = $0.5 \times 2$ = 1 mol
3) โมลอะตอมทั้งหมด = $0.5 \times (8+10+4+2)$ = $0.5 \times 24$ = 12 mol
4) อะตอมรวม = $12 \times 6.022 \times 10^{23}$ = $7.226 \times 10^{24}$ อะตอม
5) อะตอม H = $5 \times 6.022 \times 10^{23}$ = $3.011 \times 10^{24}$ อะตอม
☕ คาเฟอีนในชีวิตจริง! กาแฟ 1 ถ้วย (240 ml) มีคาเฟอีนประมาณ 95 mg = 0.00049 mol = $2.94 \times 10^{20}$ โมเลกุล นั่นคือหลายร้อยล้านล้านล้านโมเลกุลที่ทำให้ตื่น! ⚡
23K₂O 6000 โมเลกุล มีธาตุ K และ O กี่อะตอม
✦ เฉลย
K₂O = K 2 อะตอม + O 1 อะตอม ต่อ 1 โมเลกุล
อะตอม K = $6000 \times 2$ = 12,000 อะตอม
อะตอม O = $6000 \times 1$ = 6,000 อะตอม
24มีเทน CH₄ มีจำนวน H จำนวน 2800 อะตอม มาจากมีเทนกี่โมเลกุล
✦ เฉลย
CH₄ = H 4 อะตอม ต่อ 1 โมเลกุล
จำนวนมีเทน = $\dfrac{2800}{4}$ = 700 โมเลกุล
25กลูโคส C₆H₁₂O₆ จำนวน $12.04 \times 10^{23}$ โมเลกุล จะมีอะตอม H กี่อะตอม
✦ เฉลย
กลูโคส 1 โมเลกุล มี H 12 อะตอม
อะตอม H = $12 \times 12.04 \times 10^{23}$ = $1.4448 \times 10^{25}$ อะตอม
26กลูโคสจำนวน $12.04 \times 10^{23}$ โมเลกุล คิดเป็นกี่โมล
✦ เฉลย
$n = \dfrac{12.04 \times 10^{23}}{6.022 \times 10^{23}}$ = 2 โมล
29น้ำกลั่น 1800 กรัม คิดเป็นกี่โมล (H₂O = 18 g/mol)
✦ เฉลย
$n = \dfrac{g}{MW} = \dfrac{1800}{18}$ = 100 โมล
💧 น้ำ 1 แก้ว = กี่โมล? แก้วน้ำ 250 ml ≈ 250 g → n = 250/18 ≈ 13.9 โมล = $8.36 \times 10^{24}$ โมเลกุล — ดื่มน้ำทีหนึ่งได้กลืนโมเลกุล H₂O เข้าไปเกือบ 10 ล้านล้านล้านโมเลกุล! 🤯
34ยาลดกรด Magnesium hydroxide 0.04 โมล คิดเป็นกี่กรัม (Mg=24, O=16, H=1)
✦ เฉลย
MW Mg(OH)₂ = 24 + 2×(16+1) = 24 + 34 = 58 g/mol
g = n × MW = $0.04 \times 58$ = 2.32 กรัม
35แก๊ส X 0.224 ลิตรที่ STP มีจำนวนโมลเท่าใด
✦ เฉลย
$n = \dfrac{V}{22.4} = \dfrac{0.224}{22.4}$ = 0.01 โมล
37ก๊าซ CO 30 โมล คิดเป็นกี่ลูกบาศก์เซนติเมตรที่ STP
✦ เฉลย
V = n × 22.4 = 30 × 22.4 = 672 ลิตร
= 672 × 1000 ลูกบาศก์เซนติเมตร = 672,000 cm³
38แก๊ส CO₂ 88 กรัม มีกี่โมเลกุล (C=12, O=16)
✦ เฉลย
MW CO₂ = 12 + (2×16) = 44 g/mol
n = 88/44 = 2 mol
จำนวนโมเลกุล = $2 \times 6.022 \times 10^{23}$ = $1.204 \times 10^{24}$ โมเลกุล
40แก๊สแอมโมเนีย NH₃ 34 กรัม คิดเป็นกี่โมเลกุล (N=14, H=1)
✦ เฉลย
MW NH₃ = 14 + (3×1) = 17 g/mol
n = 34/17 = 2 mol
จำนวนโมเลกุล = $2 \times 6.022 \times 10^{23}$ = $1.204 \times 10^{24}$ โมเลกุล
41NH₃ รั่วไหล 0.034 g จงหา: 1) ปริมาตร 2) จำนวนโมเลกุล 3) จำนวนอะตอม N และ H
✦ เฉลย
n = 0.034/17 = 0.002 mol
1) V = $0.002 \times 22.4$ = 0.0448 ลิตร (44.8 ml)
2) โมเลกุล = $0.002 \times 6.022 \times 10^{23}$ = $1.204 \times 10^{21}$ โมเลกุล
3) อะตอม N = $0.002 \times 6.022 \times 10^{23} \times 1$ = $1.204 \times 10^{21}$
อะตอม H = $0.002 \times 6.022 \times 10^{23} \times 3$ = $3.613 \times 10^{21}$ อะตอม
🧊 NH₃ เป็นสารทำความเย็น! แอมโมเนียถูกใช้เป็นสารทำความเย็นในอุตสาหกรรมมาก่อนที่จะมีแฟรีออน (Freon) และกำลังกลับมาได้รับความนิยมอีกครั้งเพราะเป็นมิตรกับสิ่งแวดล้อมกว่า แต่มีกลิ่นฉุนมากๆ และเป็นพิษครับ!
ส่วนที่ 2 — การดุลสมการเคมี
★หลักการดุลสมการ

• จำนวนอะตอมแต่ละธาตุด้านซ้าย = ด้านขวา (กฎอนุรักษ์มวล)

• ดุลโดยการเติมสัมประสิทธิ์ (ตัวเลขหน้าสูตร) ห้ามเปลี่ยนสูตรเคมี

วิธีดุล: เริ่มจากธาตุที่ซับซ้อนที่สุดก่อน → ดุล H และ O เป็นลำดับสุดท้าย

1Fe₂O₃ + H₂ → Fe + H₂O
✦ เฉลย
Fe₂O₃ + 3H₂ → 2Fe + 3H₂O

ตรวจสอบ: Fe 2=2 ✓, O 3=3 ✓, H 6=6 ✓

🔨 นี่คือการถลุงเหล็ก! การรีดิวซ์ Fe₂O₃ ด้วย H₂ (หรือ CO) เป็นหัวใจของอุตสาหกรรมเหล็กกล้า ทุกตึก ทุกรถ ทุกสะพาน ล้วนเริ่มต้นจากปฏิกิริยานี้ครับ!
2C₂H₄ + O₂ → CO₂ + H₂O
✦ เฉลย
C₂H₄ + 3O₂ → 2CO₂ + 2H₂O

ตรวจสอบ: C 2=2 ✓, H 4=4 ✓, O 6=6+2=6 ✓

3K₂CO₃ + HNO₃ → KNO₃ + H₂O + CO₂
✦ เฉลย
K₂CO₃ + 2HNO₃ → 2KNO₃ + H₂O + CO₂

ตรวจสอบ: K 2=2 ✓, N 2=2 ✓, H 2=2 ✓, C 1=1 ✓, O 9=6+1+2=9 ✓

4Fe₂(SO₄)₃ + KOH → K₂SO₄ + Fe(OH)₃
✦ เฉลย
Fe₂(SO₄)₃ + 6KOH → 3K₂SO₄ + 2Fe(OH)₃

ตรวจสอบ: Fe 2=2 ✓, S 3=3 ✓, K 6=6 ✓, O 12+6=12+6=18 ✓, H 6=6 ✓

5Mg(OH)₂ + HCl → MgCl₂ + H₂O
✦ เฉลย
Mg(OH)₂ + 2HCl → MgCl₂ + 2H₂O

ตรวจสอบ: Mg 1=1 ✓, Cl 2=2 ✓, H 4=4 ✓, O 2=2 ✓

💊 ยาลดกรดเกิดขึ้นได้อย่างไร? Mg(OH)₂ + 2HCl → MgCl₂ + 2H₂O — OH⁻ จาก Mg(OH)₂ ไปสะเทิน H⁺ จากกรดในกระเพาะ ทำให้ pH สูงขึ้น บรรเทาอาการแสบร้อน นี่คือเคมีที่ช่วยชีวิตได้จริงๆ ครับ!
★โจทย์ก๊าซฟอสจีน (ข้อ 45)
✦ เฉลย: COCl₂ ที่ STP, V = 4.48 dm³
MW COCl₂ = 12 + 16 + (2×35.5) = 99 g/mol
1) n = V/22.4 = 4.48/22.4 = 0.2 mol
2) มวล = n × MW = 0.2 × 99 = 19.8 กรัม
3) จำนวนโมเลกุล = $0.2 \times 6.022 \times 10^{23}$ = $1.204 \times 10^{23}$ โมเลกุล
☠️ ระวัง! ก๊าซพิษสงคราม Phosgene (COCl₂) ถูกใช้เป็นอาวุธเคมีในสงครามโลกครั้งที่ 1 ทำให้เสียชีวิตมากกว่าก๊าซคลอรีน ปัจจุบันใช้ในอุตสาหกรรมพลาสติกและยา แต่มีการควบคุมอย่างเข้มงวดมากครับ
