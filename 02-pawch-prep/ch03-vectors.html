เฉลย ครั้งที่ 3

ปริมาณเวกเตอร์
แบบฝึกหัด 3.1 — ผลรวมเวกเตอร์ทิศเดียวกัน
แนวคิด:A และ B ทิศเดียวกัน → บวกขนาดได้เลย
$\vec{R} = \vec{A} + \vec{B} = 2 + 3$
$|\vec{R}| = 5$ หน่วย ทิศเดียวกับ $\vec{A}$ และ $\vec{B}$
💡 เวกเตอร์ทิศเดียวกันบวกกันง่ายที่สุด — แค่บวกตัวเลข เหมือนเดินหน้าต่อกันเลยครับ!
แบบฝึกหัด 3.2 — บวก/ลบเวกเตอร์กราฟิก
แนวคิด:บวก/ลบเวกเตอร์กราฟิกด้วยวิธีต่อหัวต่อหาง (head-to-tail)
$\vec{A} + \vec{B}$
วาง A ก่อน แล้วต่อ B ที่หัวลูกศร A → ผลลัพธ์คือเส้นจากหางของ A ถึงหัวของ B
$\vec{A} - \vec{B}$
เปลี่ยน B เป็น −B (กลับทิศ) แล้วบวกปกติ: A + (−B)
$\vec{A} + \vec{B} - \vec{C}$
ต่อ A → B → (−C) ตามลำดับ → ผลลัพธ์คือเส้นจากจุดเริ่มถึงปลายสุด
💡 วิธีกราฟิกใช้ตรวจสอบทิศทางได้ดีมาก — ในทางปฏิบัติวิศวกรใช้วิธีแยกองค์ประกอบ x,y แทนครับ
แบบฝึกหัด 3.3 — เวกเตอร์สองตัวตั้งฉากกัน
แนวคิด:เวกเตอร์สองตัวตั้งฉากกัน (90°) → ใช้ทฤษฎีพีทาโกรัส
ขนาดลัพธ์:$R = \sqrt{A^2 + B^2} = \sqrt{5^2 + 12^2} = \sqrt{25 + 144} = \sqrt{169}$
$R = 13$ หน่วย
ทิศของลัพธ์:$\theta = \arctan\!\left(\dfrac{12}{5}\right) \approx 67.4°$ จากเวกเตอร์ขนาด 5
💡 5-12-13 เป็น Pythagorean triple อีกชุดหนึ่ง เหมือน 3-4-5 ครับ — จำไว้ใช้ตรวจคำตอบได้เลย!
แบบฝึกหัด 3.4 — แรงสองแรงทำมุม 60°
แนวคิด:แรงสองแรงทำมุม 60° → ใช้กฎ cosine (parallelogram law)
สูตร:$R^2 = A^2 + B^2 + 2AB\cos\theta$
แทนค่า:$R^2 = 7^2 + 8^2 + 2(7)(8)\cos 60° = 49 + 64 + 112 \times 0.5 = 169$
$R = \sqrt{169} = 13$ N
💡 ได้ 13 N อีกแล้ว! 7-8 ไม่ใช่ triple แต่มุม 60° ทำให้คำตอบออกมาสวยงาม — ข้อนี้ออกแบบมาให้ตอบสวยครับ 😄
แบบฝึกหัด 3.5 — หาขนาดแรงลัพธ์
แนวคิด:max เมื่อทิศเดียวกัน, min เมื่อทิศตรงข้าม → หาขนาดแต่ละแรง
หาขนาดแรง A และ B
$A + B = 140 \quad , \quad A - B = 20$
$A = 80 \ \mathrm{N} \quad , \quad B = 60 \ \mathrm{N}$
เมื่อทำมุม 90° ต่อกัน
$R = \sqrt{A^2 + B^2} = \sqrt{80^2 + 60^2} = \sqrt{6400 + 3600} = \sqrt{10000}$
$R = 100$ N
💡 80-60-100 = 4-3-5 × 20 — อีก Pythagorean triple! และตรงกับที่โจทย์บอกไว้ว่า 100 N ✓ ถือว่าตรวจสอบตัวเองได้เลยครับ
แบบฝึกหัด 3.6 — หาขนาดเวกเตอร์ทั้งสอง
แนวคิด:R ตั้งฉากกับเวกเตอร์ที่เล็กกว่า (B) → R·B = 0 → (A+B)·B = 0
ตั้งสมการ
เงื่อนไข 1:$A + B = 16$
เงื่อนไข 2 (R⊥B):$\vec{R} \perp \vec{B} \Rightarrow R^2 = A^2 - B^2$
$(A+B)(A-B) = 64 \Rightarrow 16(A-B) = 64 \Rightarrow A - B = 4$
แก้สมการ
$A + B = 16 \quad , \quad A - B = 4$
$A = 10$ หน่วย $\quad , \quad B = 6$ หน่วย
💡 เงื่อนไข "R ตั้งฉากกับ B" ทำให้ได้สมการพิเศษ R² = A² − B² — ถ้าไม่เห็นตรงนี้โจทย์จะแก้ไม่ได้เลยครับ!
