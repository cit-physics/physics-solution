โจทย์ข้อ 1
แนวคิด
สูตร:$\theta(\mathrm{rad}) = \theta(°) \times \frac{\pi}{180}$
สูตรกลับ: องศา = เรเดียน × (180/π)
คำตอบ
1) 60°$= 60 \times \frac{\pi}{180} = \frac{\pi}{3} \approx 1.047 \ \mathrm{rad}$
2) 30°$= 30 \times \frac{\pi}{180} = \frac{\pi}{6} \approx 0.524 \ \mathrm{rad}$
3) 245°$= 245 \times \frac{\pi}{180} = \frac{49\pi}{36} \approx 4.276 \ \mathrm{rad}$
4) 3π rad$= 3\pi \times \frac{180}{\pi} = 540°$
5) π/2 rad$= \frac{\pi}{2} \times \frac{180}{\pi} = 90°$
6) 5π/3 rad$= \frac{5\pi}{3} \times \frac{180}{\pi} = 300°$
โจทย์ข้อ 2
แนวคิด: 1 รอบ = 2π rad, 1 นาที = 60 วินาที
สมการ:$\omega = n \times \frac{2\pi}{60}$
แทนค่า:$\omega = 1200 \times \frac{2\pi}{60} = 40\pi \approx 125.66 \ \mathrm{rad/s}$
คำตอบ:    $\omega = 40\pi \approx 125.66 \ \mathrm{rad/s}$
โจทย์ข้อ 3
แนวคิด: prefix SI
G (giga) = 10⁹, M (mega) = 10⁶, k (kilo) = 10³
m (milli) = 10⁻³, n (nano) = 10⁻⁹, p (pico) = 10⁻¹²
คำตอบ
1) 1.60 GHz$= 1.60 \times 10^9 \ \mathrm{Hz} = 1.60 \times 10^6 \ \mathrm{kHz}$
2) 2.0 ns$= 2.0 \times 10^{-9} \ \mathrm{s} \div 10^3 = 2.0 \times 10^{-12} \ \mathrm{ks}$
3) 3×10⁻³ mm$= 3 \times 10^{-3} \times 10^{-3} \ \mathrm{m} = 3 \times 10^{-6} \ \mathrm{m} = 3 \times 10^{-12} \ \mathrm{Mm}$
4) 2×10⁴ m$= \frac{2 \times 10^4}{10^3} \ \mathrm{km} = 20 \ \mathrm{km}$
โจทย์ข้อ 4
แนวคิด: kilo (k) = 10³
แทนค่า:$2{,}000 \ \mathrm{N} = 2 \times 10^3 \ \mathrm{N} = 2 \ \mathrm{kN}$
คำตอบ:    $2 \ \mathrm{kN}$
โจทย์ข้อ 5
แนวคิด: 1 ลิตร = 1,000 cm³ และ 1 cm = 10⁻² m
แปลง 1 ลิตร:$= 1{,}000 \ \mathrm{cm^3} = 1{,}000 \times (10^{-2} \ \mathrm{m})^3 = 1{,}000 \times 10^{-6} \ \mathrm{m^3} = 10^{-3} \ \mathrm{m^3}$
5 ลิตร:$= 5 \times 10^{-3} \ \mathrm{m^3}$
คำตอบ:    $5 \times 10^{-3} \ \mathrm{m^3}$
โจทย์ข้อ 6
แนวคิด: 1 km = 1,000 m และ 1 h = 3,600 s
สมการ:$v(\mathrm{m/s}) = v(\mathrm{km/h}) \times \frac{1{,}000}{3{,}600} = \frac{v}{3.6}$
แทนค่า:$v = \frac{72}{3.6} = 20 \ \mathrm{m/s}$
คำตอบ:    $20 \ \mathrm{m/s}$
โจทย์ข้อ 7
แนวคิด: T (tera) = 10¹², G (giga) = 10⁹ ดังนั้น 1 TB = 10³ GB
แปลง:$2 \ \mathrm{TB} = 2 \times 10^3 \ \mathrm{GB} = 2{,}000 \ \mathrm{GB}$
คำตอบ:    $2{,}000 \ \mathrm{GB}$
โจทย์ข้อ 8
แนวคิด: milli (m) = 10⁻³ ดังนั้น 1 mg = 10⁻³ g = 10⁻⁶ kg
แทนค่า:$250 \ \mathrm{mg} = 250 \times 10^{-6} \ \mathrm{kg} = 2.5 \times 10^{-4} \ \mathrm{kg}$
คำตอบ:    $2.5 \times 10^{-4} \ \mathrm{kg}$
โจทย์ข้อ 9
แนวคิด: 1 cm = 10⁻² m ดังนั้น 1 cm² = (10⁻²)² m² = 10⁻⁴ m²
แทนค่า:$20 \ \mathrm{cm^2} = 20 \times 10^{-4} \ \mathrm{m^2} = 2 \times 10^{-3} \ \mathrm{m^2}$
คำตอบ:    $2 \times 10^{-3} \ \mathrm{m^2}$
โจทย์ข้อ 10
แนวคิด: แปลงตัวเศษและตัวส่วนแยกกัน
ตัวเศษ:$1 \ \mathrm{g} = 10^{-3} \ \mathrm{kg}$
ตัวส่วน:$1 \ \mathrm{cm^3} = (10^{-2})^3 \ \mathrm{m^3} = 10^{-6} \ \mathrm{m^3}$
แทนค่า:$13.6 \ \frac{\mathrm{g}}{\mathrm{cm^3}} = 13.6 \times \frac{10^{-3}}{10^{-6}} \ \frac{\mathrm{kg}}{\mathrm{m^3}} = 13.6 \times 10^3 \ \frac{\mathrm{kg}}{\mathrm{m^3}}$
คำตอบ:    $13{,}600 \ \mathrm{kg/m^3}$
โจทย์ข้อ 11
แนวคิด: micro (μ) = 10⁻⁶
แปลง:$0.00005 \ \mathrm{A} = 5 \times 10^{-5} \ \mathrm{A}$
เป็น μA:$= \frac{5 \times 10^{-5}}{10^{-6}} \ \mu\mathrm{A} = 50 \ \mu\mathrm{A}$
คำตอบ:    $50 \ \mu\mathrm{A}$
โจทย์ข้อ 12
แนวคิด: หน่วยพื้นที่ไทย
1 ตร.วา = 4 m²
1 งาน = 100 ตร.วา = 400 m²
1 ไร่ = 4 งาน = 400 ตร.วา = 1,600 m²
แทนค่า
2 ไร่:$2 \times 1{,}600 \ \mathrm{m^2} = 3{,}200 \ \mathrm{m^2}$
1 งาน:$1 \times 400 \ \mathrm{m^2} = 400 \ \mathrm{m^2}$
50 ตร.วา:$50 \times 4 \ \mathrm{m^2} = 200 \ \mathrm{m^2}$
รวม:$3{,}200 + 400 + 200 = 3{,}800 \ \mathrm{m^2}$
คำตอบ:    $3{,}800 \ \mathrm{m^2}$
โจทย์ข้อ 13
แนวคิด: mega (M) = 10⁶
แปลง:$60{,}000{,}000 \ \mathrm{W} = 6 \times 10^7 \ \mathrm{W} = \frac{6 \times 10^7}{10^6} \ \mathrm{MW} = 60 \ \mathrm{MW}$
คำตอบ:    $60 \ \mathrm{MW}$
โจทย์ข้อ 14
กำหนดให้
$P = 2{,}200 \ \mathrm{MW} = 2.2 \times 10^9 \ \mathrm{W}$
$t = 24 \times 30 = 720 \ \mathrm{h} = 720 \times 3{,}600 \ \mathrm{s} = 2.592 \times 10^6 \ \mathrm{s}$
ข้อ 1: พลังงานในหน่วย จูล (J)
สมการ:$E = P \times t$
แทนค่า:$E = 2.2 \times 10^9 \ \mathrm{W} \times 2.592 \times 10^6 \ \mathrm{s}$
$E = 5.7024 \times 10^{15} \ \mathrm{J}$
คำตอบ:    $E \approx 5.70 \times 10^{15} \ \mathrm{J}$
ข้อ 2: พลังงานในหน่วย ยูนิต (kWh)
สมการ:$E = P(\mathrm{kW}) \times t(\mathrm{h})$
แทนค่า:$E = 2{,}200{,}000 \ \mathrm{kW} \times 720 \ \mathrm{h} = 1.584 \times 10^9 \ \mathrm{kWh}$
คำตอบ:    $1.584 \times 10^9 \ \mathrm{kWh}$
ข้อ 3: รายได้จากการขายไฟ (ราคา 4.20 บาท/ยูนิต)
แทนค่า:$\mathrm{รายได้} = 1.584 \times 10^9 \times 4.20 \approx 6.653 \times 10^9 \ \mathrm{บาท}$
คำตอบ:    $\approx 6{,}652.8 \ \mathrm{ล้านบาทต่อเดือน}$
โจทย์ข้อ 15
กำหนดให้
อัตราสิ้นเปลือง = 27.8 km/L, ระยะทาง = 300 km, ราคาน้ำมัน = 45 บาท/ลิตร
สมการที่ใช้
ปริมาณน้ำมัน:$V = \frac{\mathrm{ระยะทาง}}{\mathrm{อัตราสิ้นเปลือง}} = \frac{300}{27.8} \approx 10.79 \ \mathrm{ลิตร}$
ค่าน้ำมัน:$= 10.79 \times 45 \approx 485.6 \ \mathrm{บาท}$
คำตอบ:    $\approx 485.6 \ \mathrm{บาท}$
โจทย์ข้อ 16
ก) ปริมาณถ่านหินที่เทียบเท่าพลังงานจาก U-235 จำนวน 1 kg
พลังงาน U-235:$E_{U} = 8 \times 10^{13} \ \mathrm{J}$
พลังงานถ่านหิน:$E_{coal} = 24 \ \mathrm{MJ/kg} = 24 \times 10^6 \ \mathrm{J/kg}$
มวลถ่านหิน:$m = \frac{8 \times 10^{13}}{24 \times 10^6} \approx 3.33 \times 10^6 \ \mathrm{kg} = 3{,}333 \ \mathrm{ตัน}$
รถบรรทุก 1 คัน บรรทุกได้ 20 ตัน:$n = \frac{3{,}333}{20} \approx 167 \ \mathrm{คัน}$
คำตอบ:    ต้องใช้ถ่านหิน $\approx 3.33 \times 10^6 \ \mathrm{kg}$ หรือ $3{,}333 \ \mathrm{ตัน}$ ใช้รถบรรทุก 167 คัน
ข) ความยาวรวมของขบวนรถบรรทุก
รถ 1 คัน ยาว 15 m, จำนวน 167 คัน:$L = 167 \times 15 \ \mathrm{m} = 2{,}505 \ \mathrm{m}$
คำตอบ:    ความยาวรวม $= 2{,}505 \ \mathrm{m} \approx 2.505 \ \mathrm{km}$
โจทย์ข้อ 17 (ก) — เวกเตอร์ขนาด 4 หน่วย ทำมุม 30°
แนวคิด: แยกเวกเตอร์เป็นองค์ประกอบ x และ y
สมการ:$r_x = |r|\cos\theta \quad , \quad r_y = |r|\sin\theta$
แทนค่า r_x:$r_x = 4\cos 30° = 4 \times \frac{\sqrt{3}}{2} = 2\sqrt{3} \approx 3.46$
แทนค่า r_y:$r_y = 4\sin 30° = 4 \times \frac{1}{2} = 2$
เขียนในระบบพิกัดฉาก
$\vec{r} = 2\sqrt{3}\ \hat{e}_x + 2\ \hat{e}_y$
คำตอบ:    $r_x = 2\sqrt{3} \approx 3.46 \quad , \quad r_y = 2$
โจทย์ข้อ 17 (ข) — เวกเตอร์ขนาด 5 หน่วย ทำมุม 37°
แนวคิด: ใช้ sin37° = 0.6, cos37° = 0.8 (ค่าประมาณมาตรฐาน)
แทนค่า r_x:$r_x = 5\cos 37° = 5 \times 0.8 = 4$
แทนค่า r_y:$r_y = 5\sin 37° = 5 \times 0.6 = 3$
เขียนในระบบพิกัดฉาก
$\vec{r} = 4\ \hat{e}_x + 3\ \hat{e}_y$
คำตอบ:    $r_x = 4 \quad , \quad r_y = 3$
โจทย์ข้อ 18 — ขนาดและทิศทางของเวกเตอร์ลัพธ์
แนวคิด: แยกแต่ละแรงเป็นองค์ประกอบ x และ y แล้วรวม (ใช้ sin53°=0.8, cos53°=0.6)
F₁ = 30 N (+x):$F_{1x} = +30 \ \mathrm{N} \quad , \quad F_{1y} = 0$
F₂ = 40 N (+y):$F_{2x} = 0 \quad , \quad F_{2y} = +40 \ \mathrm{N}$
F₃ = 50 N (53° ใต้ -x):$F_{3x} = -50\cos 53° = -50 \times 0.6 = -30 \ \mathrm{N}$
$F_{3y} = -50\sin 53° = -50 \times 0.8 = -40 \ \mathrm{N}$
รวมองค์ประกอบ
ΣFₓ:$= 30 + 0 + (-30) = 0$
ΣF_y:$= 0 + 40 + (-40) = 0$
ขนาดลัพธ์:$|F| = \sqrt{(\Sigma F_x)^2 + (\Sigma F_y)^2} = \sqrt{0^2 + 0^2} = 0 \ \mathrm{N}$
คำตอบ:    เวกเตอร์ลัพธ์ $= 0 \ \mathrm{N}$ (ระบบอยู่ในสมดุล — แรงลัพธ์เป็นศูนย์)
โจทย์ข้อ 19 — การบวก/ลบเวกเตอร์ในระบบ 3 มิติ
กำหนดให้
$\vec{A} = 3\hat{e}_x - 2\hat{e}_y + 5\hat{e}_z$
$\vec{B} = 2\hat{e}_x - \hat{e}_y$
$\vec{C} = \hat{e}_x + 2\hat{e}_y - \hat{e}_z$
$\vec{D} = 3\hat{e}_z - 2\hat{e}_x$
(ก) หา A + B
x:$3 + 2 = 5$
y:$-2 + (-1) = -3$
z:$5 + 0 = 5$
คำตอบ:    $\vec{A} + \vec{B} = 5\hat{e}_x - 3\hat{e}_y + 5\hat{e}_z$
(ข) หา A - C + 2B
2B:$2\vec{B} = 4\hat{e}_x - 2\hat{e}_y$
x:$3 - 1 + 4 = 6$
y:$-2 - 2 + (-2) = -6$
z:$5 - (-1) + 0 = 6$
คำตอบ:    $\vec{A} - \vec{C} + 2\vec{B} = 6\hat{e}_x - 6\hat{e}_y + 6\hat{e}_z$
